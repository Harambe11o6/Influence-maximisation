# Influence Maximization on Facebook Network

This project implements **Influence Maximization** techniques on a Facebook social network dataset using Python.
The goal is to identify the most influential nodes (users) in a network that can maximize information spread.

The implementation is designed to run seamlessly on **Google Colab** or a local Jupyter environment.

## ğŸ“Œ Problem Statement
Influence Maximization is the task of selecting a small set of seed nodes in a social network such that their influence spread is maximized under diffusion models like:
- Independent Cascade (IC)
- Linear Threshold (LT)

This problem is widely used in:
- Viral marketing
- Social media analysis
- Recommendation systems
- Network science research
